# SynapseAI Medication Tracker

A mobile-first medication tracker prototype built with React and Vite.

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open the local URL shown in the terminal to preview the app.

## Project structure

- `src/main.jsx` — app entry point
- `src/App.jsx` — main app logic and navigation
- `src/styles.css` — global app styles
- `src/data/mockData.js` — mock user and medication data
- `src/components/` — reusable UI components

This prototype includes onboarding, returning user login, a dashboard, Shelf, Calendar, Insights, and Profile pages.

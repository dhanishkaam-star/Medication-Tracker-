# Synapse AI Medication Tracker

A static medication management web app built for tracking medication logs, side effect patterns, doctor visit preparation, family medication management, and cost savings.

## Features

- Medication logging with mood, energy, sleep, and symptoms
- Historical data storage using browser localStorage
- AI-style insight generation with confidence indicators
- Weekly trend charts and monthly health summaries
- Doctor visit report generation and PDF export
- Family profile management and caregiver alerts
- Cost tracking, annual projection, and savings recommendations

## Run locally

1. Open `index.html` in your browser.
2. Use the app to log medication, create reports, and manage family profiles.

## Notes

This implementation is a static front-end prototype with no backend. It is optimized for quick demonstration and local data persistence.
